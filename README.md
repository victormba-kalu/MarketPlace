# Marketplace

A modern web platform connecting small business owners (chefs, bakers, caterers, fashion designers, farmers) with service professionals (UI/UX designers, digital marketers, web developers, business analysts, etc.).

## Features

- Responsive design with dark/light mode toggle
- Mobile-friendly navigation with hamburger menu
- Dynamic provider cards fetched from RandomUser.me API
- Live search and advanced filters (category, price range, minimum rating)
- Interactive modal with provider details and client reviews
- Smooth animations and hover effects

## Technologies Used

- **HTML5**, **CSS3**, **Vanilla JavaScript** (no frameworks)
- **Tailwind CSS** (via Vite)
- **Vite** for fast development and building
- **Two Third-Party APIs**:
  - RandomUser.me – for realistic provider data and images
  - JSONPlaceholder – for fake client reviews
